### 🧭 总结常用移动命令

| **目标位置** | **命令**      | **说明**                       |     |
| -------- | ----------- | ---------------------------- | --- |
| **文件末端** | **`G`**     | 移动到文件最后一行的第一个非空白字符。          |     |
| **文件开始** | **`gg`**    | 移动到文件第一行的第一个非空白字符。           |     |
| **指定行号** | **`{行号}G`** | 移动到指定的行号。例如，`42G` 移动到第 42 行。 |     |
| **当前行末** | **`$`**     | 移动到当前光标所在行的行尾。               |     |
| **当前行首** | **`0`**     | 移动到当前光标所在行的行首（第一个字符）。        |     |

要使用这些命令，您必须处于 **普通模式（Normal Mode）**。
